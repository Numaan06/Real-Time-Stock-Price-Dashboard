Real-Time Stock Dashboard 📈

A real-time stock market dashboard built using Python and Streamlit that provides live stock prices, interactive charts, technical indicators, USD to INR conversion, and portfolio tracking features.

🚀 Features
Real-time stock market data
Candlestick and line charts
Technical indicators:
SMA (Simple Moving Average)
EMA (Exponential Moving Average)
RSI (Relative Strength Index)
USD to INR currency conversion
Portfolio Tracker
Historical stock data table
Interactive visualizations using Plotly
Live stock prices in sidebar
Multiple time-period analysis
🛠️ Technologies Used
Python
Streamlit
Pandas
Plotly
Yahoo Finance API
ta
📦 Installation

Clone the repository:

git clone <your-github-repo-link>
cd stock-dashboard

Install dependencies:

pip install -r requirements.txt

Run the project:

streamlit run stock_dashboard.py
📊 Project Output

The dashboard displays:

Live stock prices
Interactive charts
Technical indicators
Historical stock data
Portfolio profit/loss tracking
Currency conversion values
